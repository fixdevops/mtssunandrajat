# Logo dan Asset Instructions

## Logo yang Dibutuhkan

Silakan download atau buat logo berikut dan simpan di folder `public/`:

### 1. logo_putih.png
- Logo SMA Negeri 1 Yogyakarta dengan warna putih
- Ukuran: 200x200px (atau lebih besar)
- Format: PNG dengan background transparan
- Digunakan untuk: Navbar saat di atas hero section

### 2. logo_biru.png
- Logo SMA Negeri 1 Yogyakarta dengan warna biru
- Ukuran: 200x200px (atau lebih besar)
- Format: PNG dengan background transparan
- Digunakan untuk: Navbar saat di-scroll

### 3. favicon.png
- Icon kecil untuk browser tab
- Ukuran: 32x32px atau 64x64px
- Format: PNG
- Bisa menggunakan versi simplified dari logo utama

### 4. id.png
- Icon bendera Indonesia
- Ukuran: 20x20px
- Format: PNG
- Merah putih horizontal

## Alternatif Sementara

Jika belum ada logo, Anda bisa:

1. **Download dari website asli**: https://sman1yogya.sch.id/assets/images/
   - logo_putih.png
   - logo_biru.png
   - favicon.png

2. **Gunakan placeholder sementara**: 
   - Buat file PNG sederhana dengan text "SMA 1"

3. **Generate online**:

   - Gunakan https://favicon.io untuk membuat favicon

## Cara Menambahkan Logo

1. Download atau buat logo sesuai spesifikasi di atas
2. Simpan file di folder `public/`
3. Pastikan nama file sesuai:
   - `logo_putih.png`
   - `logo_biru.png`
   - `favicon.png`
   - `id.png`
4. Restart development server jika sedang berjalan

## Catatan

- Hapus file `.txt` placeholder setelah menambahkan logo asli
- Logo harus memiliki background transparan untuk hasil terbaik
- Ukuran file sebaiknya < 100KB untuk performa optimal
